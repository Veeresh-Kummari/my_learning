STACK USING LINKED LIST (AUTO MODE)
------------------------------------

This program implements a Stack using a linked list.
All operations are fully automated using random values and random operations.

No manual user input is needed except selecting the number of auto cycles.

FEATURES:
---------
1. Auto push (random numbers)
2. Auto pop
3. Auto peek
4. Auto display
5. Auto empty check
6. Auto full check (always NOT full)
7. Random operation selection
8. Memory-safe operations

EDGE CASES HANDLED:
-------------------
- Underflow detection
- Peek on empty
- Display empty stack
- Memory allocation check failure
- Push/pop cycles with safe termination
- Random sequence testing

FILES:
------
main.c
functions.c
header.h
Makefile

HOW TO RUN:
-----------
make
./stack_ll_auto

clean:
make clean
