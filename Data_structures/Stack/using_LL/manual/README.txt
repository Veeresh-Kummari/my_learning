STACK USING LINKED LIST (MANUAL MODE)
--------------------------------------

This program implements a Stack using a singly linked list.
All operations are performed manually using a menu.

FEATURES:
---------
1. Manual push
2. Pop
3. Peek
4. Display
5. Check empty
6. Linked list stack (never gets full)
7. Input validation for:
   - Characters
   - Symbols
   - Floats
   - Mixed input
   - Invalid menu input

EDGE CASES HANDLED:
-------------------
- Stack underflow
- Empty stack peek
- Display empty list
- Memory allocation failure
- Repeated invalid input
- Large integer input
- Negative integer input

FILES:
------
main.c
functions.c
header.h
Makefile

HOW TO RUN:
-----------
make
./stack_ll_manual

clean:
make clean
