STACK USING ARRAY (AUTO MODE)
------------------------------

This program implements a Stack using an array and performs all operations
AUTOMATICALLY using random values and random operations.

No manual operations are required.

FEATURES:
---------
1. Auto push (random numbers)
2. Auto pop
3. Auto peek
4. Auto display
5. Auto empty/full check
6. Random selection of operations
7. Random stack size (5â€“15)
8. No user input required except number of cycles

EDGE CASES HANDLED:
-------------------
- Auto overflow detection
- Auto underflow detection
- Display on empty stack
- Peek on empty stack
- Pushing until full
- Popping until empty
- Safe termination after cycles

FILES:
------
main.c
functions.c
header.h
Makefile

HOW TO RUN:
-----------
make
./stack_auto

clean:
make clean
